Задание 1. Создание собственного класса
Создате класс Car, у которого есть:
свойство Brand (марка автомобиля);
свойство Speed (текущая скорость);
метод Accelerate(), увеличивающий скорость на 10.

Пример:
var car = new Car { Brand = "BMW" };
car.Accelerate();
Console.WriteLine($"{car.Brand} едет со скоростью {car.Speed} км/ч");

Задание 2. Использование конструктора
Добавьте конструктор, принимающий марку автомобиля и начальную скорость:
var car = new Car("Audi", 50);

Задание 3. Класс Book
Создайте класс Book:
свойства: Title, Author, Pages;
метод Read(int pages) — выводит сообщение "Вы прочитали X страниц из Y" и не позволяет прочитать больше, чем доступно.

Задание 4. Класс Library
Создайте класс Library, который:
содержит список книг (List<Book>);
имеет методы AddBook(Book book) и PrintAllBooks().

Пример:
var lib = new Library();
lib.AddBook(new Book("1984", "Дж. Оруэлл", 350));
lib.PrintAllBooks();
