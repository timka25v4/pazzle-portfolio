Задание 2
Создайте класс Phone, у которого есть:
свойство Model — модель телефона
свойство Battery — текущий заряд (от 0 до 100)
метод Charge(int amount) — увеличивает заряд на указанное число
метод Use(int amount) — уменьшает заряд на указанное число

Пример использования:
var phone = new Phone { Model = "Samsung Galaxy" };

phone.Charge(30);
Console.WriteLine($"Заряд: {phone.Battery}%");

phone.Use(10);
Console.WriteLine($"Заряд: {phone.Battery}%");
